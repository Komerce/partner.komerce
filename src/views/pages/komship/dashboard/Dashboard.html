Tambahkan Produk

Ikuti langkah di bawah ini :
1. Pergi ke menu Produk.
2. Lengkapi from untuk menambah Produk kamu.
3. Jangan lupa untuk memeriksa ulang data yang telah dimasukan.
4. Jika sudah, lalu klik simpan.
Selamat, kamu berhasil menambahkan produk pertama kamu. Di menu produk kamu dapat memantau penjualan setiap produk dan melihat produk paling laris.

Membuat Orderan

Ikuti langkah di bawah ini :
1. Pergi ke menu Orderku.
2. Pilih tanggal orderan kamu, tanggal ini digunakan untuk menandai orderan kamu.
3. Pilih produk yang ingin kamu buat ordernya.
4. Klik lanjutkan.
5. Lengkapi form pengiriman orderan.
6. Cek kembali form pengirimannya.
7. klik Submit, Untuk membuat orderan.
Orderan kamu berhasil dibuat! Langkah selanjutnya adalah ajukan pickup agar paket orderanmu dijemput kurir.

Ajukan Pickup Orderan

Ikuti langkah di bawah ini :
1. Pergi ke menu Pickup dan klik Ajukan Pickup
2. Pilih gudang yang sudah kamu tentukan untuk pengiriman orderan.
3. Tentukan waktu pickup.
4. Pilih jenis kendaraan pickup.
5. Pilih Orderan mana yang ingin kamu pickup.
6. Lalu klik Ajukan Pickup
Ajukan Pickup berhasil dan kamu melakukan label di menu Riwayat Pickup. Pastikan semua orderan sudah siap untuk dikirim yaa!
